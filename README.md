# Gamanza-task

To start app you need to:

0. git clone git@github.com:DordeK/Gamanza-task.git
1. cd gamanza-task
2. cd api
3. npm i
4. npm run start
5. cd ..
6. cd fe
7. npm i
8. npm run start
9. open localhost:3000/
